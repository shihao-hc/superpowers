# 背景图片设置说明

## 如何设置默认背景图

1. 将您提供的图片保存为 `default-bg.jpg`
2. 将图片文件放置在以下目录：
   ```
   D:\龙虾\shihao-video\public\images\default-bg.jpg
   ```

## 图片要求
- 格式：JPG/JPEG
- 推荐尺寸：1920x1080 或更高
- 文件大小：建议小于2MB

## 当前设置
- 背景图片已配置为：`/images/default-bg.jpg`
- 背景覆盖效果：70%透明度遮罩
- 背景固定：滚动时背景保持固定
- 自适应：图片会自动缩放适配屏幕

## 自定义背景
您可以在设置页面上传自定义背景图片：
1. 访问：http://localhost:3000/setting.html
2. 在"外观设置"部分上传图片
3. 点击"应用背景"按钮
